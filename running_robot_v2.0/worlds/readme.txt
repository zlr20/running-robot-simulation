本环境为大赛虚拟标准赛 C-P的2.0竞赛环境，与1.0相比：
1、改进了计分系统，当机器人通过起点横杆后开始计时，到达终点后停止计时。
2、增加了随机地图机制。本环境在每次载入时会随机改变关卡顺序（开始与结束关卡固定），并随机生成地雷的相对位置。
3、修改了场景中部分物件的尺寸，使其与标准赛基本一致。
4、增加了关卡结尾处的水平开横杆。
5、每次比赛结束时会输出比赛过程的记录。
6、修改basicTimeStep不会再影响横杆的运动频率。





This is the 2.0 competition environment of simulation standard competition C-P

1. The scoring system is improved. When the robot passes through the starting crossbar, it starts timing and stops when it reaches the end point.

2. Random map system is added. This environment will randomly change the level order (the start and end levels are fixed). 

3. The size of some objects in the scene was modified to make it consistent with the standard competition.

4. Added a cross bar at the end of the level.

5. At the end of each game, a record of the game will be output.

6. Modifying basictimestep node will no longer affect the movement frequency of the crossbars.
